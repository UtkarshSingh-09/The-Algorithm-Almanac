## ğŸš€ Features

* **Context-Aware Answers:** Uses RAG to fetch specific passages from the book before answering.
* **Smart Query Expansion:** Automatically expands simple queries (e.g., "SGD") into full questions to improve search results.
* **Verbatim Guardrails:** If a user asks a question that matches a known definition exactly, the system returns the direct text from the knowledge base.
* **Source Transparency:** API responses include metadata on whether the answer was generated by AI or retrieved directly from the text.
* **High Performance:** Built on FastAPI for asynchronous, fast request handling.
